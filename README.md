# UCAS Knowledge Engineering

本仓库整理了中国科学院大学（UCAS）知识工程课程的复习资料，内容主要基于 2026 年春季学期课程 PPT 简化、归纳而成，用于期末复习和知识点快速回顾。

由于课程内容、PPT 和考试重点每年可能会有所调整，请以当年授课材料为准。本仓库资料仅作为学习参考。

## 资料优势

- 内容覆盖较为全面，尽量涵盖课程 PPT 中的主要知识点和核心概念。人工校验覆盖大部分课程的重点内容。
- 大部分内容在保留原文表述和关键术语的基础上，大幅压缩篇幅，减少复习时的信息负担与考试中的耗时（考试的体验，在基于本文档考试找到所有答案的前提并基于精简后的内容回答的前提下，时间依然只是勉强够用）。
- 提供思维导图，方便根据关键词快速定位相关知识模块，并梳理章节之间的联系。

## 仓库内容

- `知识工程复习资料.md`：完整复习资料，包含课程主要章节内容和思维导图。
- `分章节资料/`：按章节拆分后的 Markdown 文档，便于单独打印成册。

目前分章节资料包括：

1. 知识工程概述
2. 认知的基石：记忆、知识及其表征
3. 语义网与知识图谱
4. 大模型认知能力分析与评测
5. 大模型知识机理：知识溯源与表征分解
6. 大模型知识编辑与遗忘
7. 大模型推理增强
8. 智能体规划
9. 智能体的工具和记忆
10. 智能体环境与世界模型

## 使用建议

- 对于大部分的考题可以直接找到相对应的总结内容，已经大幅精简原文内容更适合直接作为解答。
- 由于整体篇幅较多，考前还是建议对照的文档整体阅读一遍，了解文档的大体内容，考试中可以通过知识图谱的关键词来进行查找。

## 贡献与更正

如果你发现资料中存在错误、遗漏、表述不清，或希望补充新的课程内容，欢迎提交 Pull Request。

提交 PR 时建议说明：

- 修改了哪些章节或知识点；
- 修改原因，例如 PPT 更新、原文错误、表达优化或内容补充；
- 如有依据，可附上对应课程材料页码或参考来源。

当前26春考题仍然等待补充，欢迎任何形式的更正与完善！
